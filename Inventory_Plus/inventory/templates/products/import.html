Import Products from CSV

CSV Format Requirements

Your CSV file should have the following columns:

  • name - Product name (required)
  • sku - Stock Keeping Unit (required, unique)
  • category - Category name (will be created if doesn't exist)
  • description - Product description
  • unit_price - Price per unit
  • quantity_in_stock - Current stock quantity
  • reorder_level - Minimum stock level
  • suppliers - Comma-separated supplier names
Sample CSV Format
name,sku,category,description,unit_price,quantity_in_stock,reorder_level,suppliers
Laptop Dell XPS,LAP001,Electronics,High performance laptop,1299.99,15,5,TechWorld Inc
Office Chair,CHA001,Furniture,Ergonomic office chair,299.99,8,3,Furniture Pro
